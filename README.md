# MyInTime
간단한 Django blog 프로젝트 입니다.

## Features
- 블로그 메인 화면 : 부트스트랩 사용
- 사용자 블로그 글 쓰기
- CKEditor : 글쓰기 페이지 꾸미기
- 블로그 세부 화면 구성
- 댓글 추가
- MyInTime 카운트

## Installation
```
pip install -r requirements.txt
cd pjt_MyInTime
python manage.py runserver
```
